# Carpeta de Imágenes

Coloca aquí las imágenes que uses en la aplicación.

## Logo de la Empresa

Para que el logo aparezca en la página de inicio, coloca el archivo **Koncilia.png** en esta carpeta.

### Instrucciones:

1. Coloca el archivo `Koncilia.png` en esta carpeta: `wwwroot/images/`
2. El logo se mostrará automáticamente en la página de inicio
3. Si la imagen no se encuentra, se mostrará un icono por defecto

### Ubicación del archivo:

```
Koncilia_Contratos/
└── wwwroot/
    └── images/
        └── Koncilia.png  ← Coloca tu imagen aquí
```
